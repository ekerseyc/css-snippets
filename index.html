Mini Project - CSS Snippets

This is a short description about CSS cheatsheets.

Flexbox Row

Use these properties to create a Flexbox row layout.

                
.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use these properties to create a Flexbox column.

                
.column {
    display: flex;
    flex-direction: column;
}

Box Shadow

Use these properties to add a blurred shadow to any box.

                
.box {
    border: 1px solid;
    padding: 10px;
    box-shadow: 5px 10px 18px #888;
}

Linear Gradient

Use these properties to create a linear gradient from top to bottom.

                
.linear-gradient-bg {
background-image: linear-gradient(
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102, 236, 0.6) 100%
    );
}

Box Transition Glow

Use these properties to create a glowing border when hovering.

                
.code-card, .code-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}

.code-card:hover, code-card:hover
.card-header {
    box-shadow: inset 0px 0px 8px
    rgba(232, 102, 236, 1); 0 0 15px
    rgba(232, 102, 236, 1);
}

Overlay Card with Title

Use this to make a card similar to what you see on this page!

                
.card-header {
    position: relative;
    margin-top: -20px;
}
